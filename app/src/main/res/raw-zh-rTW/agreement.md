# OpenMonitor 使用者協議

最後更新：2026年4月12日

歡迎使用 **OpenMonitor**。請在使用本應用前仔細閱讀以下條款。

## 1. 條款接受

使用 OpenMonitor 即表示您同意受本條款約束。如果您不同意，請勿使用本應用。

## 2. 應用簡介

OpenMonitor 是一款開源的 Android 系統監控工具，提供即時硬體狀態、FPS 幀率錄製、進程管理等系統監控功能。

## 3. 權限使用

OpenMonitor 可能根據您選擇的執行模式請求以下權限：

- **Root 權限**：用於完整的系統資訊存取
- **Shizuku**：無需 Root 即可執行特權操作
- **懸浮窗**：用於懸浮監控視窗顯示
- **使用量統計**：用於電池和應用使用統計
- **通知**：用於前台服務通知

所有權限僅用於系統監控目的，資料均在您的裝置本地處理。

## 4. 資料收集與隱私

- 所有監控資料均在**您的裝置本地**收集和儲存
- 不會向外部伺服器傳輸任何個人資料
- Firebase Analytics 可能收集匿名使用統計以協助改進應用
- 您可隨時在系統設定中停用分析功能

## 5. 開源聲明

OpenMonitor 是開源軟體，您可以在 GitHub 上查看完整原始碼。本應用使用了多個開源程式庫，各程式庫受其相應授權條款約束。

- 開發者不對因使用本應用造成的任何損害承擔責任
- Root 和 Shizuku 操作存在固有風險，請自行斟酌使用
