# OpenMonitor 使用者協議

最後更新：2026年4月12日

歡迎使用 **OpenMonitor**。請在使用本應用前仔細閱讀以下條款。

## 1. 條款接受

使用 OpenMonitor 即表示您同意受本條款約束。如果您不同意，請勿使用本應用。

## 2. 應用簡介

OpenMonitor 是一款開源的 Android 系統監控工具，提供即時硬體狀態、FPS 幀率錄製、進程管理等系統監控功能。

## 3. 權限使用

OpenMonitor 可能根據您選擇的執行模式請求以下權限：

- **Root 權限**：用於完整的系統資訊存取
- **Shizuku**：無需 Root 即可執行特權操作
- **adb 模式**：透過 adb 連線取得系統資訊
- **位置權限**：僅用於取得 Wi-Fi 相關資訊
- **懸浮窗**：用於懸浮監控視窗顯示
- **使用量統計**：用於電池和應用使用統計
- **通知**：用於前台服務通知

所有權限僅用於系統監控目的，資料均在您的裝置本地處理。

通常來說，Root 和 Shizuku 模式提供更全面的監控功能，而 adb 模式則適合不希望授予過多權限的使用者。您可以根據需要選擇合適的執行模式。

無論是何種模式，OpenMonitor *不會對系統進行任何的修改、注入等危險操作*，任何操作均為唯讀模式。在保證系統安全的前提下提供最佳的監控體驗。

## 4. 資料收集與隱私

- 所有監控資料均在**您的裝置本地**收集和儲存
- 不會向外部伺服器傳輸任何個人資料
- Firebase Analytics 可能收集匿名使用統計、崩潰日誌以協助改進應用，但**不會包含任何個人資訊**
- 我們將取得您裝置的唯一識別碼（如 Android ID 等類似技術手段）僅用於區分裝置、避免濫用，但**不會與任何個人資訊關聯**

- adb 模式需要開啟開發者選項和 USB 偵錯，請確保了解相關風險
