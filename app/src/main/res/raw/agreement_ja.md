# OpenMonitor 利用規約

最終更新：2026年4月12日

**OpenMonitor** をご利用いただきありがとうございます。本アプリケーションを使用する前に、以下の規約をよくお読みください。

## 1. 規約への同意

OpenMonitor を使用することにより、本規約に同意したものとみなされます。同意されない場合は、本アプリケーションを使用しないでください。

## 2. アプリケーションの概要

OpenMonitor は、リアルタイムのハードウェア状態、FPS 録画、プロセス管理などのシステム監視機能を提供するオープンソースの Android システムモニターツールです。

## 3. 権限の使用

OpenMonitor は、選択した実行モードに応じて以下の権限を要求する場合があります：

- **Root アクセス**：完全なシステム情報へのアクセス
- **Shizuku**：Root なしでの特権操作の実行
- **adb モード**：adb 接続を介してシステム情報を取得
- **位置情報**：Wi-Fi 関連情報の取得にのみ使用
- **オーバーレイ**：フロート監視ウィンドウの表示
- **使用状況統計**：バッテリーとアプリの使用状況統計
- **通知**：フォアグラウンドサービス通知の表示

すべての権限はシステム監視の目的でのみ使用され、データはデバイス内でローカルに処理されます。

一般的に、Root と Shizuku モードはより包括的な監視機能を提供し、adb モードは過度な権限を付与したくないユーザーに適しています。必要に応じて適切な実行モードを選択できます。

どのモードであっても、OpenMonitor は*システムに対していかなる変更、インジェクション等の危険な操作も行いません*。すべての操作は読み取り専用であり、システムの安全性を確保しながら最高の監視体験を提供します。

## 4. データ収集とプライバシー

- すべての監視データは**デバイス内にローカルで**収集・保存されます
- 個人データが外部サーバーに送信されることはありません
- Firebase Analytics がアプリ改善のために匿名の使用統計やクラッシュログを収集する場合がありますが、**個人情報は一切含まれません**
- デバイスの固有識別子（Android ID 等の類似技術手段）をデバイスの区別および不正使用防止の目的でのみ取得しますが、**いかなる個人情報とも関連付けられません**

## 5. オープンソース

OpenMonitor は複数のオープンソースライブラリを使用しており、各ライブラリはそれぞれのライセンスに従います。「設定」-「オープンソースライセンス」で確認できます。

- Root および Shizuku の操作には固有のリスクがあります。ご自身の判断でご利用いただき、リスクはご自身でご負担ください
- adb モードには開発者オプションと USB デバッグの有効化が必要です。関連するリスクをご理解ください
