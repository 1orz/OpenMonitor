## 2. 应用简介

OpenMonitor 是一款开源的 Android 系统监控工具，提供实时硬件状态、FPS 帧率录制、进程管理等系统监控功能。

## 3. 权限使用

OpenMonitor 可能根据您选择的运行模式请求以下权限：

- **Root 权限**：用于完整的系统信息访问
- **Shizuku**：无需 Root 即可执行特权操作
- **adb 模式**：通过 adb 连接获取系统信息
- **位置权限**：仅用于获取 Wi-Fi 相关信息
- **悬浮窗**：用于悬浮监控窗口显示
- **使用量统计**：用于电池和应用使用统计
- **通知**：用于前台服务通知

所有权限仅用于系统监控目的，数据均在您的设备本地处理。

通常来说，Root 和 Shizuku 模式提供更全面的监控功能，而 adb 模式则适合不希望授予过多权限的用户。您可以根据需要选择合适的运行模式。

无论是何种模式，OpenMonitor *不会对系统进行任何的修改，注入等危险操作*，任何操作均为只读模式。在保证系统安全的前提下提供最佳的监控体验。

## 4. 数据收集与隐私

- 所有监控数据均在**您的设备本地**收集和存储
- 不会向外部服务器传输任何个人数据
- Firebase Analytics 可能收集匿名使用统计，崩溃日志以帮助改进应用，但**不会包含任何个人信息**
- 我们将获取您设备的唯一标识符（如 Android ID等类似技术手段）仅用于区分设备，避免滥用，但**不会与任何个人信息关联**
